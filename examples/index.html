Vitologi inputtags plugin


Description

The plugin transforms ordinary form inputs, into fields, which allows easy create tags.

Simple use


unique tags (example)

Code Description 
var input = document.getElementById("input"),
    inputInputtags = new Inputtags(input);
Simple use allows you work with unique tags

Repeat tags


not unique tags (example)

Code Description 
var input = document.getElementById("input"),
    inputInputtags = new Inputtags(input,{
        "duplicate":true
    });
Repeat tags case allows you work with not unique tags

Tag objects


tag-objects (example)

Code Description 
var input = document.getElementById("input"),
    inputInputtags = new Inputtags(input,{
        "keyName":"key",
        "valueName":"value"
    });
Tags like objects will appear when in parameters will set "keyName" and "valueName" properties

Tags sources


tags sources (example)

Code Description 
var input = document.getElementById("input"),
    inputInputtags = new Inputtags(input,{
        "source":function(cursorValue){
        return ["Tag1","Tag2","Tag3","Tag4","Tag5","Tag6"];
    }
    });
Inputtags can use sources provider
